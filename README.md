A playground for my experiments with Raylib (for Phix and Euphoria)



This wrapper (binding) is still under development. The goal is to create the most complete wrapper possible for Phix on 64-bit Windows. “Under development” does not mean that it is non-functional or unstable, but rather that it is not yet complete. The best way to see the progress of development is to look at the \[examples](https://github.com/andizk4kx/raylib-playground/tree/main/examples). These also serve as documentation for using \[RayLib](https://github.com/raysan5/raylib) with Phix.



This wrapper (binding) is largely based on the work of Icy\_Viking (https://github.com/gAndy50/EuRayLib5)
